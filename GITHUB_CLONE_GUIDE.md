https://github.com/your-username/taiwan-bingo-app.git\ncd taiwan-bingo-app\n```\n\n### 步驟 2：檢查系統要求\n\n```bash\n# 檢查 Node.js 版本（需要 18.x 或更高）\nnode --version\n\n# 檢查 npm 版本\nnpm --version\n\n# 如果未安裝 pnpm，運行以下命令\nnpm install -g pnpm\npnpm --version\n```\n\n### 步驟 3：安裝依賴\n\n```bash\npnpm install\n```\n\n### 步驟 4：配置環境變數\n\n```bash\n# 複製環境變數示例文件\ncp .env.example .env.local\n\n# 編輯 .env.local 填入必要的環境變數\n# 最重要的變數：\n# - DATABASE_URL: MySQL 連接字符串\n# - VITE_APP_ID: Manus OAuth 應用 ID\n# - JWT_SECRET: JWT 密鑰（最少 32 字符）\n```\n\n### 步驟 5：初始化數據庫\n\n```bash\n# 生成遷移文件\npnpm run db:generate\n\n# 執行數據庫遷移\npnpm run db:migrate\n\n# （可選）填充測試數據\npnpm run db:seed\n```\n\n### 步驟 6：啟動應用\n\n```bash\n# 開發模式\npnpm run dev\n\n# 或生產模式\npnpm run build\npnpm run start\n```\n\n## ⚡ 一鍵啟動\n\n### Linux/Mac\n\n```bash\nbash scripts/setup.sh\n```\n\n該腳本會自動：\n1. ✅ 檢查 Node.js 和 pnpm\n2. ✅ 安裝依賴\n3. ✅ 配置環境變數\n4. ✅ 初始化數據庫\n5. ✅ 構建應用\n\n### Windows\n\n```bash\nscripts\\setup.bat\n```\n\n## 🐳 Docker 部署\n\n### 使用 Docker Compose（推薦）\n\n```bash\n# 1. 複製環境變數文件\ncp .env.example .env\n\n# 2. 編輯 .env 文件配置環境變數\nnano .env\n\n# 3. 啟動容器\ndocker-compose up -d\n\n# 4. 查看日誌\ndocker-compose logs -f app\n\n# 5. 停止容器\ndocker-compose down\n```\n\n### 使用 Docker 單獨構建\n\n```bash\n# 構建鏡像\ndocker build -t taiwan-bingo-app .\n\n# 運行容器\ndocker run -d \\\n  --name taiwan-bingo \\\n  -p 3000:3000 \\\n  -e DATABASE_URL=\"mysql://user:password@host:3306/db\" \\\n  -e VITE_APP_ID=\"your_app_id\" \\\n  -e JWT_SECRET=\"your_secret\" \\\n  taiwan-bingo-app\n```\n\n## 🔧 環境變數配置\n\n### 必需變數\n\n| 變數名 | 說明 | 示例 問題 1：Node.js 版本不兼容\n\n**症狀**：`pnpm install` 失敗\n\n**解決方案**：\n```bash\n# 檢查 Node.js 版本\nnode --version\n\n# 需要 18.x 或更高版本\n# 使用 nvm 更新 Node.js\nnvm install 22\nnvm use 22\n```\n\n### 問題 2：數據庫連接失敗\n\n**症狀**：`Error: connect ECONNREFUSED`\n\n**解決方案**：\n```bash\n# 檢查 MySQL 是否運行\nmysql -u root -p -e \"SELECT 1;\"\n\n# 檢查 DATABASE_URL 是否正確\ngrep DATABASE_URL .env.local\n\n# 確保數據庫存在\nmysql -u root -p -e \"CREATE DATABASE IF NOT EXISTS taiwan_bingo;\"\n```\n\n### 問題 3：端口被佔用\n\n**症狀**：`Error: listen EADDRINUSE :::3000`\n\n**解決方案**：\n```bash\n# Linux/Mac：找到佔用端口的進程\nlsof -i :3000\n\n# 殺死進程\nkill -9 <PID>\n\n# 或使用不同的端口\nPORT=3001 pnpm run dev\n```\n\n### 問題 4：環境變數未加載\n\n**症狀**：`undefined` 環境變數\n\n**解決方案**：\n```bash\n# 確保 .env.local 文件存在\nls -la .env.local\n\n# 檢查文件內容\ncat .env.local\n\n# 重新啟動應用\npnpm run dev\n```\n\n### 問題 5：依賴版本衝突\n\n**症狀**：`peer dependency` 警告或錯誤\n\n**解決方案**：\n```bash\n# 清除 pnpm 緩存\npnpm store prune\n\n# 重新安裝依賴\nrm -rf node_modules pnpm-lock.yaml\npnpm install\n```\n\n## 📞 獲取幫助\n\n- 📖 查看完整文檔：`cat DEPLOYMENT_GUIDE.md`\n- 🐛 報告 Bug：https://github.com/your-username/taiwan-bingo-app/issues\n- 💬
