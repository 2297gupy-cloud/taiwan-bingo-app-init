utf8mb4_unicode_ci\n- **ORM**: Drizzle ORM\n- **遷移工具**: Drizzle Kit\n\n### 表清單\n\n| 表名 | 說明 | 記錄數 |\n|------|------|--------|\n| `users` | 用戶表 | 可變 |\n| `drawRecords` | 開獎記錄 | 數千 |\n| `aiStarPredictions` | AI一星預測 | 數千 |\n| `aiSuperPrizePredictions` | AI超級獎預測 | 數千 |\n| `verificationRecords` | 驗證記錄 | 數百 |\n| `stats` | 統計數據 | 數百 |\n\n## 📊 表結構\n\n### users - 用戶表\n\n```sql\nCREATE TABLE users (\n  id VARCHAR(36) PRIMARY KEY,\n  email VARCHAR(255) NOT NULL UNIQUE,\n  name VARCHAR(255),\n  role ENUM('user', 'admin') DEFAULT 'user',\n  createdAt BIGINT NOT NULL,\n  updatedAt BIGINT NOT NULL,\n  INDEX idx_email (email),\n  INDEX idx_role (role)\n) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;\n```\n\n| 列名 | 類型 | 說明 | 約束 |\n|------|------|------|------|\n| `id` | VARCHAR(36) | 用戶 ID | PRIMARY KEY |\n| `email` | VARCHAR(255) | 郵箱 | NOT NULL, UNIQUE |\n| `name` | VARCHAR(255) | 用戶名 | - |\n| `role` | ENUM | 角色 (user/admin) | DEFAULT 'user' |\n| `createdAt` | BIGINT | 創建時間戳 | NOT NULL |\n| `updatedAt` | BIGINT | 更新時間戳 | NOT NULL |\n\n### drawRecords - 開獎記錄表\n\n```sql\nCREATE TABLE drawRecords (\n  id VARCHAR(36) PRIMARY KEY,\n  drawNumber VARCHAR(20) NOT NULL UNIQUE,\n  drawTime BIGINT NOT NULL,\n  numbers JSON NOT NULL,\n  superNumber INT,\n  bigSmall ENUM('big', 'small'),\n  oddEven ENUM('odd', 'even'),\n  plate VARCHAR(50),\n  source VARCHAR(50),\n  timestamp BIGINT NOT NULL,\n  createdAt BIGINT NOT NULL,\n  INDEX idx_drawNumber (drawNumber),\n  INDEX idx_drawTime (drawTime),\n  INDEX idx_timestamp (timestamp)\n) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;\n```\n\n| 列名 | 類型 | 說明 | 約束 |\n|------|------|------|------|\n| `id` | VARCHAR(36) | 記錄 ID | PRIMARY KEY |\n| `drawNumber` | VARCHAR(20) | 期數 | NOT NULL, backup.sql\n\n# 壓縮備份\nmysqldump -u username -p database_name | gzip > backup.sql.gz\n\n# 備份特定表\nmysqldump -u username -p database_name table_name > backup_table.sql\n```\n\n### 恢復數據庫\n\n```bash\n# 恢復完整備份\nmysql -u username -p database_name < backup.sql\n\n# 恢復壓縮備份\ngunzip < backup.sql.gz | mysql -u username -p database_name\n\n# 恢復特定表\nmysql -u username -p database_name < backup_table.sql\n```\n\n### 定期備份腳本\n\n```bash\n#!/bin/bash\n# backup.sh - 每日自動備份\n\nBACKUP_DIR=\"/backups/taiwan-bingo\"\nDB_NAME=\"taiwan_bingo\"\nDB_USER=\"backup_user\"\nDB_PASS=\"backup_password\"\n\nDATE=$(date +%Y%m%d_%H%M%S)\nmysqldump -u $DB_USER -p$DB_PASS $DB_NAME | gzip > $BACKUP_DIR/backup_$DATE.sql.gz\n\n# 保留最近 30 天的備份\nfind $BACKUP_DIR -name \"backup_*.sql.gz\" -mtime +30 -delete\n```\n\n## 🔍 常用查詢\n\n### 查看表大小\n\n```sql\nSELECT \n  TABLE_NAME,\n
