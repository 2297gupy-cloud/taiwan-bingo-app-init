# UI 配置指南\n\n## 📋 目錄\n\n- [色彩主題](#色彩主題)\n- [字體配置](#字體配置)\n- [響應式設計](#響應式設計)\n- [組件庫](#組件庫)\n- [自定義樣式](#自定義樣式)\n- [深色模式](#深色模式)\n\n## 🎨 色彩主題\n\n### 全局色彩變數\n\n編輯 `client/src/index.css` 修改全局色彩：\n\n```css\n@layer base {\n  :root {\n    /* 亮色主題 */\n    --background: 0 0% 100%;          /* 背景色 */\n    --foreground: 0 0% 3.6%;          /* 前景色 */\n    --primary: 262 80% 50%;           /* 主色 */\n    --primary-foreground: 210 40% 98%; /* 主色前景 */\n    --secondary: 217 32% 17%;         /* 次色 */\n    --secondary-foreground: 210 40% 98%; /* 次色前景 */\n    --destructive: 0 84% 60%;         /* 危險色 */\n    --destructive-foreground: 210 40% 98%; /* 危險色前景 */\n    --muted: 220 13% 91%;             /* 柔和色 */\n    --muted-foreground: 217 33% 17%;  /* 柔和色前景 */\n    --accent: 217 91% 60%;            /* 強調色 */\n    --accent-foreground: 210 40% 98%; /* 強調色前景 */\n    --border: 220 13% 91%;            /* 邊框色 */\n    --input: 220 13% 91%;             /* 輸入框色 */\n    --ring: 262 80% 50%;              /* 聚焦環色 */\n    --radius: 0.5rem;                 /* 圓角半徑 */\n  }\n\n  .dark {\n    /* 深色主題 */\n    --background: 0 0% 3.6%;\n    --foreground: 0 0% 98%;\n    --primary: 262 80% 50%;\n    --primary-foreground: 220 14% 96%;\n    /* ... 其他深色主題變數 */\n  }\n}\n```\n\n### 色彩使用示例\n\n```tsx\n// 使用語義化色彩類名\n<div className=\"bg-background text-foreground\">背景和文本</div>\n<button className=\"bg-primary text-primary-foreground\">主按鈕</button>\n<div className=\"border border-border\">邊框</div>\n<div className=\"bg-destructive text-destructive-foreground\">危險操作</div>\n```\n\n### 色彩值格式\n\n應用使用 OKLCH 色彩空間（推薦）：\n\n```\nformat: hue saturation lightness\nexample: 262 80% 50%\n```\n\n**轉換 Hex 到 OKLCH**：\n\n```javascript\n// 使用在線工具：https://oklch.com/\n// 或使用 CSS 函數：\ncolor: oklch(50% 0.2 262);\n```\n\n## 🔤 字體配置\n\n### 添加 Google Fonts\n\n編輯 `client/index.html`：\n\n```html\n<head>\n  <!-- Noto Sans TC - 繁體中文 -->\n  <link
